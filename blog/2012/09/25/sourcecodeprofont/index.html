Source Code Pro を Mac で試す。

Sep 25th, 2012

Adobeがコーディング用のソースコードをオープンソースで公開している。 9/24 に開催されたCreate The Webで発表された。

Adobe公式ブログにも記事がアップされている。 Announcing Source Code Pro

提供は、Sourceforge, GitHub, Adobe Edge Web Fonts, Typekit, WebINK, Google Web Fontsから入手可能。今回は馴染み深い Github から取得してみようと思う。

今回の環境

iMac Mountain Lion

GitHub を確認

adobe / Source-Code-Pro を見てみるとREADMEが充実しているので順番に実施していく。

※ いろいろやってみたがここでは結果のみを記載

Pre-build font binaries

ここから ダウンロードして適当な場所にコピー。

解凍後、フォント一覧がでてくるので~/Library/Fontsにコピーすると該当ユーザのみ使用可能となる。

フォントをビルドしようと試行錯誤してみたがうまくいかないので上記方法で終了。 失敗記録は別に書いてみようか。
